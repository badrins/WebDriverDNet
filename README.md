  MAKE.ID as ID,
  MAKE.NAME,
  MAKE.MAKECODE,
  MAKE.MODELCODE,
  SHOWROOM.PRIZE

  CAR MAKE JOIN CAR MODEL on MODEL.ID=MAKE.PARENTCARID JOIN SHOWROOM on MAKE.ID=SHOWROOM.CARDID
